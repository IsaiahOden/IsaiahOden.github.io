Weekly Learning Summary

This week, I expanded my knowledge in various areas of web development. Below are some key concepts I focused on:

Syntax

Syntax was arguably the most important term and concept we discussed this week. It's essentially the structure of the code and how it's written. Syntax is crucial because it can make or break the functionality of a program.

Git Push

Now this was one of the terms that we probably had to use most often in our assignments this week. Git push is essentially the command that sends your committed changes to a remote repository. It's a crucial step in the version control process.

Version Control

A crucial concept that will persist throughout our entire career in technology, version control is usually controlled through GitHub. It's essentially a way to manage changes to code over time, and it's crucial for collaboration and tracking changes.

Domain

The same way we call each other by our given name rather than our Social Security Number, domains allow us to access websites by name rather than by IP address. It's a pivotal part of the internet and web development.

HTML

HTML stands for HyperTextMarkupLanguage and is the first language that we learn in Code School. It's the backbone of most websites on the internet and is used to structure content on the web. Understanding HTML is fundamental to web development.

Growth Mindset

The Growth Mindset was the most powerful topic we covered last week in class. It's the idea that our abilities and intelligence can be developed with time, effort, and dedication. This concept is crucial for success in any field.

Git Commit

Another one of the Git commands we used frequently this week was Git commit. This command is used to save changes to the local repository.
